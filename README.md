# ansible_role_podman

Install libcontainers tools relating to Podman (buildah, podman, and skopeo).


## Requirements

None.


## Role Variables

None.


## Dependencies

None.


## Example Playbook

```
---
- name: Install podman tools on all hosts
  hosts: all
  roles:
    - name: ansible_role_podman
```
